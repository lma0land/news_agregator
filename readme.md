# 📰 News Aggregator Project

## 📌 Описание проекта
News Aggregator — это групповой учебный fullstack проект, который собирает новости из различных источников, обрабатывает их, классифицирует по категориям и отображает пользователю в удобном интерфейсе.

Цель проекта — автоматизировать сбор и структурирование новостной информации с использованием скрапинга, обработки текста и веб-интерфейса.

---

## 🏗️ Архитектура системы

Проект построен по клиент-серверной архитектуре и включает следующие основные компоненты:


Frontend <--> Backend (Flask API) <--> Database (SQLite)
|
v
News Processing Pipeline


### 🔄 Pipeline обработки данных:

Источники новостей
↓
Скрапинг (BeautifulSoup)
↓
Очистка данных (cleaning)
↓
Классификация (TF-IDF / ML модель)
↓
Сохранение в базу данных (SQLite)
↓
API (Flask)
↓
Frontend (HTML/CSS/JS)


---

## 👥 Роли участников команды

Перед началом разработки команда распределила обязанности для эффективной работы над проектом.

| Роль | Ответственность | Примеры задач |
|------|----------------|---------------|
| **Backend Developer** | Разработка серверной части, сбор и хранение данных | Скрапинг новостей с помощью `BeautifulSoup`, работа с `SQLite`, создание структуры базы данных, обработка данных перед сохранением |
| **ML Engineer** | Обработка текста и классификация новостей | Реализация TF-IDF, обучение модели для определения категории новости (спорт, политика, технологии и т.д.), улучшение точности классификации |
| **Fullstack Developer** | Связь frontend и backend, разработка интерфейса | Разработка API на `Flask`, создание UI (HTML/CSS/JS), реализация поиска, фильтрации и отображения новостей |

---

## ⚙️ Основной функционал

- Сбор новостей из различных источников
- Автоматическая классификация по категориям
- Удаление дубликатов
- Поиск по ключевым словам
- Фильтрация по категориям
- Отображение новостей в интерфейсе
